# STM32C5 demo workshop

In this 3-hour workshop, ST experts will introduce the STM32C5 microcontroller series and show how it redefines entry‑level and mainstream designs by combining high performance, rich integration, and cost efficiency.
Powered by an Arm® Cortex®‑M33 core and ST’s 40 nm flash technology, the [STM32C5 series](https://www.st.com/en/microcontrollers-microprocessors/stm32c5-series.html) delivers up to 144 MHz and 593 CoreMark, offering up to three times the performance of typical Cortex‑M0+–based devices at an attractive cost point.
Beyond hardware, the workshop highlights how the upgraded STM32Cube ecosystem, including STM32CubeMX2, STM32 HAL2, and STM32CubeIDE for VS Code, helps developers evaluate, configure, and develop faster, accelerating time‑to‑market and reducing development effort.
Practical examples also illustrate how existing STM32 projects can smoothly transition to STM32C5 while benefiting from the latest toolchain improvements.

### What you will learn

- Understand the STM32C5 architecture, features, and positioning. Be able to assess whether it fits your application or customer project requirements
- Get started quickly with STM32C5 using the upgraded STM32Cube ecosystem
- Explore developer tips and best practices using STM32CubeMX2, STM32 HAL2, and STM32CubeIDE for Visual Studio Code
- Interact directly with ST experts and get immediate answers to technical or commercial questions
